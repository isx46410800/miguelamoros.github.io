Comandos lenguaje MARKDOWN

  • Títulos: #,##...
  • Underlines: ------/====
  • Negrita: **/__
  • Cursiva: */_
  • Tachado ~~
  • Lista normal: */-
  • Lista numerada: 1. / 2. ...
  • Líneas para encabezado después de título: ===/---
  • Notas: >
  • Código de bloque: ``
  • Bloque de texto: ```tipo_lenguaje
  • Líneas de separación: ***/---/___
  • Link: titulo[]()web
  • Link automático: <>
  • Imagen: titulo![]()ruta imagen
  • Tablas: | letra | letra / --- linea
  • Crear indice: 1. [Primer apartado](#id1) -- ## Primer apartado
